BehaveN
-------

There is no installer. Extract the assemblies to some folder and add a
reference to BehaveN.dll from your test project.

BehaveN is free to use for both free and commercial projects. See
LICENSE.txt for the actual license.

For the list of changes, see HISTORY.txt.

The project wiki on Google Code has plenty of examples and documentation:

http://code.google.com/p/behaven

For the API details, see BehaveN.chm.

If you have any questions or suggestions, feel free to email me:

mailto:jason@diamond.name

Thanks and happy coding!
